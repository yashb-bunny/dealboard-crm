Google OAuth Setup 🔧
Follow these steps to enable Google sign-in for Dealboard.
Step 1 — Google Cloud Console
1. Go to console.cloud.google.com
2. Create a project → APIs & Services → Credentials
3. Create OAuth 2.0 Client ID (Web application)
4. Add Authorised redirect URI:
https://dijaqnbfqsajbfniozwd.supabase.co/auth/v1/callback
5. Copy your Client ID and Client Secret
Step 2 — Supabase Dashboard
1. Go to Authentication → Providers
2. Enable Google
3. Paste your Client ID and Client Secret
4. Save changes
Step 3 — Phone OTP (Twilio)
1. Go to Authentication → Providers → Phone
2. Enable Phone provider → Select Twilio
3. Add your Twilio Account SID, Auth Token, and SMS number
4. Save — OTP will be sent via SMS instantly ✅
